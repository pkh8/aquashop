# 수족관 쇼핑몰 프로젝트

## 개발 환경
- 'Java 11'
- 'JDK 11.0.17'
- 'IDE : IntelliJ'
- 'Framework : Springboot
- 'Database : MySQL
- 'ORM : JPA'
